Factory Procedure for Noise Floor Calibration

You must have a quiet and access to a controlled “quiet” environment to complete this procedure.

  1. Place the system inside the controlled environment. This can be a quiet area or box.
  2. Attach the is attached and configured to the system.
  3. In the calibration panel, make sure the combination values are the same as those shown in the image below. DO NOT deviate from these settings unless your system needs to be fine tuned to suit your site’s requirements and reduce the time taken to calibrate each box.
  4. Navigate to the Noise Floor Bounding Tab and make sure that the calibration values are set as shown below.

Default Noise Floor Bounding Calibration Values

  1. Select the Collect training data button. This will cause the system to iterate through all the combined settings and collect suitable training data on each. Expect this to run for 6-8 hours and is best run overnight.

It is critical that the system not be interrupted until the data collection has ended. If it is interrupted, you will need to start this process again from the start.

  1. When the data collection process has completed, select the Train model button. This process examines the collected data and creates the noise floor bounds model. This only takes a few minutes.

Once the model has been trained it will automatically become available for the users. They will see this as a green "light" () in this dialog and they will then have the ability to turn the model on or off.
