Post-installation Instructions

In time these instructions will go into the Installation/Configuration Manual.

  1. If not already done, check the rear panel sensor and optical connector. If damaged, replace as required. Contact Customer Support if necessary.
  2. Connect the Sensor to the Helios Controller before the unit is switched on.

If there is no Sensor connected to the Controller, it could damage the rear panel connector when the unit is switched on!

  1. Make sure the key on the front of the panel is switched to the Laser Enable position. The Laser Enabled light will illuminate.

Pre-pack up Instructions

  1. The key on the front of the panel is switched to the Disabled position. The laser locked out light will illuminate. This makes sure that the laser can never be accidentally turned on, when there is no sensor connected to the back panel.
  2. Refer to the Maintenance Manual for instructions to clean and check the rear panel sensor and optical connector. However, if it is damaged, replace as required before the unit is next used. Contact Customer Support if necessary.
  1. Refer to the Installation Manual for guidance about how to pack-up the unit and system.
