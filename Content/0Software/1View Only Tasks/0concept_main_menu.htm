 Colour map...

The colour map is where you can configure the Detection Alarm Types. The colour map can be manipulated in order to optimize the  Graph display.

Select Main Menu > Colour Map… . This displays the Colour Map manipulation dialog, as shown in the The Colour Map manipulation dialog image. To add, remove or adjust the settings of the colour map, save and remove Colour Control Points and Colour Highlights, refer to the Colour Map Settings.

 Preferences...

From here you can edit the colour map and configure how to view information in the Graph and Waterfall displays. The properties that can be manipulated include: the Time zone, units in which measurements are recorded, the orientation and colours to be displayed in the waterfall display, playback parameters and alarm clearing intervals. You can also reset all the values to their default settings.

The Preferences Dialog

See below for a description of the options available within this dialog.
  1.  Time can be viewed in time zones. There are two zones available:
  • Local time, which is based on the time shown in your browser.
  • UTC (GMT), which is based on Greenwich Mean Time. To use this option, the Helios Controller also has to be set to UTC (GMT)
  1. The Units preferences are displayed as decimal places. You can select which units you would like to view in these measurements as well as the precision in decimal places that you would like them to be rounded to.
  2. To change the Graph orientation, select the drop-down menu and choose the desired orientation.
  3. To change the highlight colour, refer to Change the Colour Control Point & Highlight Colour.
  4. Show alarms in sound field controls whether Alarms are displayed in the main Graph window. The Alarms are displayed if:
  • An Alarm maker that sits on the baseline of the graph window.
  • A small red triangle is located along the time axis.
  • A small red triangle is located along the distance axis.
  • Alarm dot size in sound field & Alarm dot color in sound fieldcontrols change the size of the dot visible in the graph, and the colour of the graph. To display an Alarm dot, make sure that the Show alarm dot in sound field check box is selected.
  1. The Waterfall Display preferences control the orientation, and how the alarms appear, in the Graph:

  • The Alarms clearing interval is set the time that an Alarm can remain present in the Alarm list and Graph. After this time, the alarm will be discarded and will no longer visible.
