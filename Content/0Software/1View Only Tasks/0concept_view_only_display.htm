Graph

The Graph displays the most recent shot and displays the sensitivity and intensity of disturbance activity that occurs across the area that is monitored. This activity is measured in scaled values. You can alter the scale value to measure a specific signal character and zoom in/out and pan regions in the waterfall display.

 Currently Viewing

This section of the display presents the alarm detections configured for the Playback or LiveFibre logged data that is displayed the Graph and Waterfall Display.

Data logging management is located to the right of your display and allows the user to log to one of three different formats. HDF5 and NetCDF4 are common formats which work with many third party tools. FDS (Fotech Data Stream) is our proprietary format designed to allow maximum write speeds. The user can log the raw data of either the full line or a section of the line to any of these three formats. Those data files can then be used in Playback mode. Additionally the user can log the current graph (the display type). These files cannot be used for playback (as they do not contain the full raw data) but can be imported into other tools for analysis. The Data Logging Controls

These controls let you control how you mount a disk drive (refer to How to fit a Data Drive), set the data logging properties (presumably you need a section describing the data logging properties) and Start Data Logging. To do this, go to the relevant section and complete the tasks. The disk drive MUST BE mounted before logging can begin.
