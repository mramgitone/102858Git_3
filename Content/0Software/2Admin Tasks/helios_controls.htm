Processing Module

This module displays the compute load. If the load exceeds 80%, a warning icon is displayed. If this happens, reduce the compute load. To do this, reduce the PRF. This increases the FFT Size and enables Enable Frame stacking toggle & Frame stacking factor or Spatial decimation factor.

DO NOT let the compute load sit at 100% for an extended period (more than 15 seconds). The Backlog will begin to grow. It can take longer to process information and relinquish the buffers.

Logging Module

This module displays the rate at which the raw data over user-selected sections of the sensor, is written to disk. The compute load is calculated against the percentage of time the system spends physically writing data to the disk. For example, if the Logging Module spends 50% of its time waiting for data to be written, the compute load will be 50%. This metric is very closely related to the Data Write Speed.
