How to set-up the Helios Software

This procedure is to be completed by a Fotech Engineer. DO NOT attempt set-up unless you have completed the necessary training! If you need to be trained to complete this procedure, contact us.

To set-up the Helios software, cross refer to the Helios Installation & Configuration Manual, then complete these tasks:

  1. Download the software from the given Fotech location. For details, refer to the datasheet that is shipped with the product.
  2. Login as an administrator and add users to the system. Refer to Log-in Instructions
  3. Refer to Manage users... for instructions and do not forget that each person needs a login type. Refer to Types of User.
  1. Configure settings for each Alarm Reports... and Zones... to be monitored, if this has not already been done by the Fotech Field Engineer.
  2. Set-up the Communication between the Helios software and server. Go to Admin > Communications and refer to the Communications Properties and Installation & Configuration Manual.
  3. Check the Preferences..., so you can view in right time zone, and so forth.

Turn the Helios Unit on

Check the mains power and necessary cables are plugged-in, then switch on the Helios unit. Refer to the Post-installation Instructions and Installation & Configuration Manual, if you experience any problems. If you need further help, contact us.
