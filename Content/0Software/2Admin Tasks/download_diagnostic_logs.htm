Download Diagnostic Logs

If you have to report a problem with the Helios system, it helps the customer support team if current diagnostic information is available and sent to them. To download the Diagnostic logs:

Select Admin > Download Diagnostic Logs. This will download a compressed Linux tar file (*.tar) to your Downloads folder, unless the browser is configured to save files elsewhere. This file also contains a number of diagnostic log files and additional system configuration files.

Attach the downloaded tar file to an email addressed to customer.support@fotechsolutions.com. Include with the file:

  • A description of the problem.
  • How it happened.
  • Any steps you have taken to resolve the problem.

Sometimes, Technical Support can ask you to confirm the Optics Type, refer to Optics Tab for more information, and if necessary Report a problem....
