The Colour Map

The colour map is where the colours of the Waterfall and Shot displays are configured. It can be manipulated in order to optimize various items in these displays. SelectMain Menu > Colour Map…. This brings up the Colour Map manipulation dialog, as shown below.

The Colour Map manipulation dialog

The Colour Map configures the display amplitudes as colours. You do this with the creation of a number of control points and then choose the control point colour. The system will then inserts linearly the colours for all amplitudes between the control points.

Change the Scale of the Colour Map

The colour map is set to a scale based on the current Display Type. This means that each Display Type has a separate setting for the maximum scale of the Colour Map, and that maximum is changed automatically as you select different Display Types.

The Raw Shot and Mean Shot Display Types share the same scale.

You can change the maximum value for the current Display Type in two ways:

  • Enter the value you want into the text field shown in the amplitude ruler of the main display.
  • Enter the maximum value into the text field labelled “Scale”, in the Colour Map display.

Refer to Change the Colour Control Point & Highlight Colour and Colour Map Settings for information about colour highlights, to add, remove and change a colour control point and load a colour map.
