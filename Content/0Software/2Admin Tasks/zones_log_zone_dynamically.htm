Log a Zone dynamically

Global logging cannot happen concurrently. When a zone starts to log, it will disable global logging.

Select the icon. This will initiate data logging of that selected zone. If global logging is currently active, it will terminate when zone logging commences. If the system stops when the [picture] icon is selected, the zone will commence logging as soon as the system has started. This is because more than one zone can be logged at the same time.
