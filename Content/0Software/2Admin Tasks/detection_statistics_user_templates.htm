Detection Statistics User Templates

Create a User Template from a Factory Template

  1. Make sure you can see the required Template Type You will see the dialog below, with the dialog split into sections, as marked.

Please let me know if you can see differences between when you are viewing this as a factory user and View Only user.

Detection Statistics Factory Templates Dialog

  1. Select the desired Factory template in the template section. These are listed in the yellow selection box shown above.
  2. If necessary, change the properties of the Factory Template. For a description of what each property and field does, in their respective disturbance, detection and alarm set-up tabs, refer to the list below:

(Not all of these links and their content have been created yet. A link will appear once it has been done)

  • User Template Properties (Detection Statistics)
  • User Template Properties (Indicators)
  • User Template Properties (Streams)
  • User Template Properties (Alarm Reports)
  1. To confirm the changes for each property value, select either the Apply or Submit buttons, whichever appears in the dialog that accompanies the associated property. Any property with the icon allows you to define and apply properties dynamically to a live system.
  2. The Cancel button prevents the property changes, and the Reset button restores the original values.
  3. Select the Save as… button. A pop-up dialog box appears to prompt you for the name of the new User Template as shown below.

Enter New User Template Name Dialog

  1. Enter a new name and select the OK button. This will create a new User Template, and all the properties of the input Factory Template will be assigned to it. User Templates are listed in the top left of the Detection Statistics Factory Templates Dialog.

A User template can be used to create another User template in exactly the same way as a Factory template can. To do this, select the Save as… button and give the User template an alternative name.

Edit a User Template

  1. Choose Admin > Detection Statistics. You will see the dialog shown below.

The Detection Statistics Dialog

  1. Select the template from the required list on the left that you wish to change.
  2. Complete your changes to the properties in the property form.
  3. Select the Apply and Save buttons to confirm the changes.

Delete a User Template

  1. Select from the required list, the template you wish to delete.
  2. Choose the icon. A dialog appears that will prompt you to confirm deletion.
  3. Select the OK button to delete the template or the Cancel button to stop deletion of the template.
