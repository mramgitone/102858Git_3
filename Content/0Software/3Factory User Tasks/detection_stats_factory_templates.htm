Detection Statistics Factory Templates

This image shows the basic high-level process flow chart. It shows the how Helios can detect in a single Stream, the single Alarm Report, on a valid Disturbance. Each blue box signifies a step in the process.

Helios allows for simultaneous generation of multiple Alarm Reports. Multiple Alarm Reports and Streams must be defined, and to support this, multiple templates and indicator masks have to be created also.

Below is the Detection Statistics Factory Template Dialog. There are Factory and User templates available with the Helios Web Interface to enable you to configure the disturbance, detection and alarm set-up.

You must be logged in as a Factory User to create Factory Templates, and these are usually created during the installation and configuration of your system. However, Factory Templates can be used to create new User templates. Contact your System Administrator or contact us if you need a Fotech Solutions Field Engineer to create these templates for you.
