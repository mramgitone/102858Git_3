Manage Snapshots

To save a database snapshot:

  1. Select Admin > Manage snapshot. Below is the screen that is displayed.

The Manage Snapshots Dialog

Once a snapshot file has been created, a number of options become available. Snapshot files can be:

  • Restored in whole, or in part, to the same Helios Controller they were taken from.
  • They can be downloaded to your local computer.
  • Uploaded to another Helios Controller.
  • Restored in whole or in part.

You can download, upload and import all or parts of databases. The sections below give instructions for the options available, once a snapshot file has been created.

Download a Database Snapshot

Select the Download button beside the desired database. The file will be automatically downloaded into the default Downloads folder on the user’s local computer. This file can then be emailed to another user to have it uploaded to, and imported into, another Helios controller.

Upload a Database Snapshot

  1. Select the Choose File button. A standard file selection dialog appears.

<image>

  1. Navigate to the folder that contains the database snapshot file.
  2. Select the file and then the Open button in the file selection dialog. The selected file will then be listed in the Manage Snapshots dialog to the right of the Choose File button.
  3. To have the selected file from your local computer uploaded to the Helios Controller, select the Upload button. The file will then appear in the list of available snapshot files, and can be selected to import into the Helios Controller.
  4. If you do not want to upload the file, select the Cancel button. To delete a selected file, select the Delete Selected button.

Import a Database Snapshot

  1. Select the Import button that is beside the desired snapshot file. A new dialog will be displayed. It prompts the user to either restore the entire snapshot database, or to only select a specific component of the snapshot database.

<image>

  1. Select the appropriate option, then choose the Submit button to confirm your choice, the Cancel button to return to the Manage Snapshots dialog, or the Reset button to restore the database currently in use.

If you select the Submit button, the Import snapshot dialog is shown below.

<image>

  1. To completely restore the database, refer to Restore the Database.

Import an Element of a Database

As an alternative to the restoration of an entire database snapshot, individual elements of the database can be imported from the database snapshot file.

Properties in Helios are divided up into Channel properties and Alarm Reports (and their components). Channel properties include all properties that are not specific to a particular stream or Alarm Report. To import a portion of a database snapshot:

  1. Make sure the second radio button entitled Select elements to import has been selected.
  2. Select the check box for the required element of the database snapshot to import.
  3. Import the Channel Properties.
