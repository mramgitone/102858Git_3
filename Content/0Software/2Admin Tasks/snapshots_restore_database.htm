Restore the Database

  1. Make sure the top radio button entitled Completely replace current db with import db has been selected.
  2. Select the Submit button. The database will be restored and the system rebooted.
  3. Select the Cancel button to dismiss the Import dialog without the database restored, and to reboot the system. Select the Reset button to restore the system back to its previous state.

Even with a complete database restored, the Hardware Properties are never overridden. Entire databases can be copied between Helios Controllers.
