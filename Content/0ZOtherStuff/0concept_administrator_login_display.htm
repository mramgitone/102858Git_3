Data Logging Controls

Data logging management is located to the right of your dashboard and allows the user to log to one of three different formats. HDF5 and NetCDF4 are common formats which work with many third party tools. FDS (Fotech Data Stream) is our proprietary format designed to allow maximum write speeds.The user can log the raw data of either the full line or a section of the line to any of these three formats. Those data files can then be used in Playback mode. Additionally the user can log the current Graph (the display type). These files cannot be used for playback (as they do not contain the full raw data) but can be imported into other tools for analysis.

The Graph logging always logs the entire monitored portion of the .

These controls let you control how you mount a disk drive (see How to fit a Data Drive), set the data logging properties (presumably you need a section describing the data logging properties) and Start Data Logging. To do this, go to the relevant section and complete the tasks. The disk drive MUST BE mounted before logging can begin.
