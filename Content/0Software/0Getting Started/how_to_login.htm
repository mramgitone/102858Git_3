How to Log-in

Types of User

There are three types of user defined in the . Most of the time your level of access will be predetermined. The types of user are:

  • Basic user. This login is for someone who can only view events as they happen. This person will see the waterfall display only.
  • Administrator. This login is for someone who is an operator of the system and allows them to manage events as they happen and also to create and amend user logins.
  • Factory user. This login is reserved for Fotech Employees only. Only in special circumstances are these details given to a client. This login allows the database to be wiped and resets the system to its factory settings.

Login as an Operator (System Administrator)

Helios Controllers are all shipped with a pre-installed user account called Admin. The first time you log into Helios, you MUST use this account.

During the login procedure, never select the Remember this User option, as it will interfere with the login authentication process.

To access this account:

  1. Choose theMain Menu > Access Secure Pagesoption.
  2. Enter the password, which you will find on the Datasheet provided to you during the installation process.

What if my Username and Password do not work?

Once a username is created it cannot be changed, so it is likely that it is a problem with the password. If the combination of your username and password does not work, report this to your System Administrator so your password can be reset.

In the event that you are the Operator or Factory User, contact us.
