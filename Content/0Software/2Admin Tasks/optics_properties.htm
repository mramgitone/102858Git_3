Optics Properties

Below is the Optics Module Dialog and a description of each of the fields.

Optics Module Dialog

Optics type

This describes the type of optics module used in your installation. This can be either GS (General Sensitivity), HS (High Sensitivity) or HSI (High Sensitivity – Improved). You can be asked for this information if you use the Fotech Customer Support system.

Optics serial number

This is used to identify the optics module in the Helios Controller, and is for reference only.

Board type

This identifies the laser pulser board, contained in the Helios Controller. It is for reference only.

Board serial number

This gives the serial number for the laser pulser board.

Firmware version

This gives the firmware version used in the Helios Controller. It is for reference only.

Laser Drive Current

This shows the live laser drive current of the Helios Controller and MUST NOT be edited.

If this field is changed, the Helios Controller can significantly lose sensitivity! If the factory default settings are reset, this can reset every property in the Controller.

This field is included in the Factory Defaults and be restored. To do this, you need to have Factory User access. Contact your Person In Charge and/or Customer Support for more information.

Trigger Level

This shows the existing trigger level of the digital acquisition board. This property does not require further adjustment. However, if the system does not trigger on incoming shots, contact Customer Support.

Box Delay

This is the time delay for the first digitized sample of a shot to travel to the back panel of the controller. The time delay depends on the amount of to travel down. This property has been set precisely at the factory and should not be adjusted.

TEC Controller Setpoint

This is used to keep the laser optimally tuned and will very rarely require field adjustment. The TEC controller Setpoint is adjusted with the assistance of an Optical Spectrum Analyzer. For more information about this field, refer to your training materials, and if the TEC controller needs adjustment, contact Customer Support.
