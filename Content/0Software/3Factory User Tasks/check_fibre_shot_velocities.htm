Velocities

Create/Track a Velocity

  1. Make sure that the Graph display is not in zoom or pan mode. Refer to Fibre Graph and the Table of Fibre Graph Functionalities.
  2. Hold down and drag the cursor from one end of the Event Disturbance, to the other end of the event, and release. A line is displayed in the current highlight colour, and the velocity will be displayed in the line.
  3. Repeat this procedure as required.

Remove a Velocity

The Clear Velocities icon [shown here] will only appear if you have drawn velocity track lines on the Graph display. Velocities cannot cross time discontinuities in the Graph (denoted by a red line).

To remove all velocity tracks display on the Graph, select the [highlighter] icon, underneath the Play/Pause [shown here] icon.
