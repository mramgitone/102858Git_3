Operations Set-up

If Claire gets time she needs to do screenshots for this page

  1. Check the optics properties in the Optics Tab and refer to Optics Properties.
  2. Check and set the Optical Channel Properties.

Your optics channel properties dialog may look slightly different from the one shown here depending on the type of optics unit you have.

  1. Check and Display the Fibre Shot.
  2. If necessary, change the Display Scale in the top left-hand corner of the Operations Overview Screen. The Display Scale is shown in a signal to noise ratio. It can be changed in real time, and is set to a scale between 1 and 5. Refer to the The Colour Map image.
  3.  Complete the General System Checks and then check your Industry Settings.
  4. Use The Colour Map to define the colour control points to be used to identify the types of alarm you want to monitor, and the Display Scale, if you did not set the Display Scale before. Refer to Colour Map Settings and Change the Colour Control Point & Highlight Colour.
  5. Log the raw data. A disk drive must be fitted, and a directory and file name defined before you can continue! Refer to How to fit a Data Drive and to Start Data Logging, for further instructions.
  1. To playback logged data, refer to Playback Mode for instructions.

If at any time during this procedure you need to log-on/off, refer to How to Log-in and Logout and Pack-up.
