Suppression Tab

Suppression Zones are visible only in the Stream Detection Value display type. They are not visible in any of the other Graph display types.

There are situations where it can be advantageous to block the response for a subsection of the sensor. This can be due to excessive nuisance disturbances. It is for this reason that the Suppression zone module was developed. The image below shows the Suppression tab where suppression zones can be defined for the current Optical Channel. Suppression zones can affect all streams, or be defined to affect only a specific stream.

The Suppression Tab

Create a Suppression Zone

To create a suppression zone, follow this process:

  1. From the drop-down menu, select the stream type. The stream can be either Active or Inactive. Select the stream to be associated with the suppression zone that is to be configured. If all of the streams are to be associated with the suppression zone, select All Streams.
  2. Select the icon and enter the desired start and end distances for this suppression zone into the text boxes.

Suppression Zone Definition dialog

The start distance must be ≥0, and the end distance must be greater than the start distance, and less than or equal to the physical length for the current Optical Channel.

OR

Choose the Select visually button to select the Start and End distances from the Graph display.

  1. Repeat steps 1 to 3 to create additional suppression zones.
  2. Select the Apply and Save buttons to save the changes.

Change Suppression Zone

  1. From the drop-down menu, select the stream the suppression zone is associated with.
  2. Edit the Start (m) and/or End (m) distances as required.

OR

Choose the Select visually button and reselect the Start and End distances from the Graph display.

  1. Select the Apply and the Save buttons to confirm the changes.

Delete a Suppression Zone

  1. From the drop-down menu, select the stream the suppression zone is associated with.
  2. Select the icon. This will remove the selected suppression zone from the list.
  3. Select the Apply and the Save buttons to confirm the changes.
