User Template Properties (Streams)

Below is an image of the Streams User Template and a description of the fields for each section.

Streams User Template Dialog

Description of Fields

Detection Statistic

From the drop-down menu, select the input Detection Statistic template. The list is divided up into factory templates and user defined templates. It will default to the currently selected template.

Disturbance Detection Threshold

For an alarm on a specific disturbance, the detection statistic needs to be converted into an alarm indicator. Any detection statistic values above the Alarm Detection Threshold and below the Alarm Suppression Threshold will produce a raw alarm indicator.

Failure to set the Alarm Detection Threshold to the correct value will significantly reduce the accuracy of disturbance alarming.

Set the Alarm Detection Threshold at a level that will allow the system to alarm only on valid disturbances. Once the Alarm Indicators are produced, then they can be configured to remove any false positives and false negatives.

A quick way to determine an optimal Alarm Detection Threshold is to use a colour bar with a colour discontinuity at a specific position in the colour map. Refer to Configure Alarm Detection Thresholds in Streams for instructions.

As the sensor increases in length, the sensitivity of the sensor decreases. Therefore, a disturbance detected closer to the controller can have a higher amplitude than one detected further down the sensor.

In order to detect disturbances anywhere along the sensor, the Alarm Detection Threshold needs to be reduced for longer distances. Create a few zones that cover the further sections of the sensor, then override the Alarm Detection Threshold for each stream in each zone. Refer to Change the Properties of a Zone about how to override zone properties.

Enable Disturbance Suppression

Used to define an upper threshold to create a range of values that will produce Alarm Indicators. To enable this, make sure the Enable alarm suppression check box is selected.

Disturbance Suppression Threshold

Enter a value for the Alarm Suppression Threshold. Indicators are not produced when Alarm detection values are above the Alarm Suppression Threshold. This can prevent alarms on extremely high amplitude disturbances.
