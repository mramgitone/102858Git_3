Communications Properties

The Communications option details the properties required to initiate data logging and the information to activate raw streaming events, to a network server.

  1. For details about Data Logging and communications set-up, see the image below and refer to Logging Module.

The Data Logging Tab

  1. To set the Network address of the server that will receive the information, and its port number, go to the Raw Streaming tab and enter the URL for the server. This will be an 11 digit number. Enter also the port address.

The Raw Streaming Tab

  1. Make sure the Raw Streaming check box is selected.
  2. Select Apply and then Save to start streaming.
  3. To remove the values entered, select Cancel and to remove the values, select Reset.
