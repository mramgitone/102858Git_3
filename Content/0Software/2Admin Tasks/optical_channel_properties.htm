Optical Channel Properties

It can be necessary to have multiple sensors attached to a Helios controller. An optical multiplexor is used to connect sequentially, to each . As each sensor is different, it requires its own set of properties. The () can handle multiple sensors by the definition of a unique Optical Channel for each sensor.

In addition to the multiple sensors attached to a single controller, you can configure simultaneous detections and alarm types on multiple disturbance types. This requires multiple streams to process alarms on multiple disturbance types. There is a core set of properties that are defined for each optical channel. Additional properties are defined for each process stream used to satisfy the alarm requirements. The Optical Channel Properties dialog contains this core set of properties for each Optical Channel. For more details about which configurations to check, the Operator will need to view the image below and complete this process:

Core properties of the Optical Channel Properties Dialog

  1. Check the sensor attached to the back panel of your Controller. Refer to the Installation & Configuration Manual as required.
  2. Select Admin > Optical Channel Properties and check the settings for the name, attributes and zero point. The names need to be unique to each sensor.
  3. If a break occurs, an additional length of optical needs to be spliced into the sensor. If this is done, all distance correlations and zone distance definitions, will be displaced by the amount of additional being inserted. To remove this distance displacement, the user can inform HWI of the splice. To view splices whilst logging, review the history, refer to Insert a Splice.
  4. Make sure the Zero Point is set to zero on the Graph. In situations when the zero point is further down the sensor, set the Zero Point to that desired distance.

A non-zero point can only be used for the Graph display. Any distances defined in the system dialog will refer to the actual non-adjusted distances.

  1. If a Panoptes server is installed with your Helios solution, a Fotech Engineer MUST check that the Panoptes ID matches the associated Sensor on the Helios Controller. Refer to the Panoptes Installation & Configuration Manual, and contact Customer Support if you require help.
  2. Next, you need to check the settings in the tabs. Refer to the list below for instructions to check the options under each tab:
