Get Started for Internal/Factory Users

To set-up and operate the Helios System, it is advised that you attend a training course and at least know how to log data. If you need to book a training course, contact Customer Support. If necessary, refer to your training materials for more in-depth reasons why you have to complete each step. [Training links should be directed to the exact place in the slides where theory is covered.]

Below are instructions to set-up and start to use the Helios System. It is recommended that you read the whole section before you start.

  1. Have you checked you can use the system? Refer to Pre-requisites of Use and read the Guidelines for the use of the Help File?
  2. Refer to the Installation & Configuration Guide to check the system is set-up and ready to use and complete the Post-installation Instructions.
  1. Log-in to the software. You will see the Operations Overview Screen.
  2. Refer to the What’s New to see what the new functionality is for this release.
  3. Set up the Helios Software. Check the default settings.
  4. Make sure the mains switch is turned on.
  5. If you are a Factory User, you will need to complete instructions for How to Set-up the Helios Software.
  6. Turn the Helios unit on.
  7. Start the Helios software. To do this, go to the Helios Controls and change the software status. The options will turn on and the buttons will illuminate in red.
  8. Complete the Operations Set-up instructions.

If at any time during this procedure you need to log-on/off. Refer to How to Login and How to Log out.

You will need to have at least System Administrator level access to the system and get authority from your company to request a Factory User login, as these steps are normally completed by a Fotech Engineer.

Contact Customer Support if necessary, then complete the Operations Set-up instructions.
