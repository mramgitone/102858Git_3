Optics Tab

This tab controls the laser properties, which pulses the in order to produce the actual response. The specific details of what is in this tab can change from one optics unit to another, and depends on the options that were chosen when you purchased your unit. This topic describes the core properties that you will see on any system. If your system differs from this, you will have received a separate description of those differences. Laser Properties

To check the laser properties, see the image below and complete these tasks:

Optics Tab showing Laser and EDFA properties

  1. Set the Pulse Repetition Frequency (PRF). You can enter the value, use the drop-down list of predetermined common values, or the button.

An increase in the PRF will increase the response bandwidth and increase the compute load, as there will be more shots processed in the time interval.

  1. Check and set the Pulse Width in accordance to the length of the sensor. 50ns and 100ns are the common properties.

If changes are required, these need to be completed by a Fotech Engineer because knowledge of the physics used, is required. Contact Customer Support as required.

The essential tradeoff is that a longer pulse width will be more sensitive (since it sends more light down the ) but will have less spatial accuracy.

The actual pulse width is limited by a step size based on the internal hardware clock. As a result the actual pulse width will be close to, but not exactly equal to, the value you request. To get a value you can either:

  • Enter the pulse width value, and the Clock Periods value will display the number of steps with the closest value.
  • Enter the number of clock periods and it will display the pulse width.
  1. Check the Bias current (mA) field. This sets the amplitude of the light pulse that is launched by the source laser. This has a range of 188-220mA. You can enter the value, use the drop-down menu or the button. To do this, you may have to refer to your training materials.
  1. Set the Pulse Trigger Mode to one of the settings described below:

By default, this is set to Internal. This will use the internal trigger mechanism and is the normal means of operation. The result is that you will see shots when the system starts.

 

Set it to External, only if you are to use an external trigger mechanism. When it is in external trigger mode, nothing will happen until the external trigger is received.

You must also set the Operation Type when this property is used. If this field is set to Continuous, then once the initial trigger has been received the internal trigger will take over and continue to trigger at the PRF.

 

If this field is set to Timed then you must also specify the Timed Run Duration. In this mode once the external trigger is received the triggers will continue at the PRF until the time specified here has been completed. Then the system will pause and wait for the next external trigger to arrive and it will restart the process.

  1. The optical amplifiers are rarely changed from the factory defaults.

If changes are required, these need to be completed by a Fotech Engineer because knowledge of the physics used, is required. Contact Customer Support as required.
