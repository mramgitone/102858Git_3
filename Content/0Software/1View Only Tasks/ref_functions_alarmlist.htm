Alarm List Functions

Icon Description
Alarm Clearance icon

Used to clear all alarms from the Graph Alarm List.

Once Alarms are removed from the Graph, they cannot be retrieved or viewed again. Check that you want to remove the Alarms from the list before you use this function.

Alarms are also cleared automatically when they appear older than the given interval that is configured in the user preferences.
Add Filter

Used to filter out or view Alarms as they are generated. You can filter out Alarms by type, magnitude, width, or speed.

For a list of Alarm types, refer to Table of Alarm Detection Types.

Remove Filter This will remove a filter and update the Alarm list to show all current Alarms.
Restrict Filters From within the Alarm Filter dialog, you can restrict Alarm Types to only show certain events. For example, to show only one Alarm Type, only select that Alarm Type from the list.
Restrict Magnitude, Width or Speed To set-up a filter, enter in the minimum and maximum for each required property and select the Filter button. Select the Dismiss button to abort application of the filter.
