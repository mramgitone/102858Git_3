How to Set-up the Helios Software

This procedure is normally completed by a Fotech Engineer. On the odd occasion this product is rented out, a trained user can complete this procedure. If you need to be trained to complete this procedure, contact us to arrange a suitable date.

To set-up the Helios software, cross refer to the Helios Installation & Configuration Manual, then complete these tasks:

  1. Download the software from the given Fotech location. Refer to the datasheet that is shipped with the product for details.
  2. Login as appropriate and add users to the system. Refer to Manage Users Properties for instructions and do not forget that each person needs to be either a View Only User or Administrator. For more information, refer to Types of User.

Do we need to go into more detail about the Alarm Reports? I have a spare topic in place if we need it, or is linking to the Properties enough?

  1. Configure settings for each type of Alarm and Zone to be monitored, if this has not already been done by the Fotech Field Engineer. Refer to Alarm Report Properties, User Template Properties (Alarm Reports) and Zone Properties, as required.
  2. Set-up the Communication between the Helios software and server. Go to Admin > Communications and refer to the Communications Properties and Installation & Configuration Manual.
  3. Check Reporting preferences in the Hardware Properties option, so you can view in right time zone, and so forth.
