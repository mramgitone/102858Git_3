Logout and Pack-up

  1. Stop the Helios Software. To do this, go to the Operations Overview Screen > Helios Controls, and refer to the Change the Monitoring Status to turn off the laser and software.
  2. When you have finished a job then you can log off. To do this, go to Main Menu > Log out.

Example of log out from Admin Mode

If you are in Administrator or Factory mode, Main Menu > Log out returns you to the basic operations display (View Only mode). If you need to log back in, refer to the Log-in Instructions. Note that you do not need to log out if you are exiting your browser anyway. Typically the only time you need to do this is if you wish to login again as a different user.

  1. Refer to the Installation & Configuration Manual for instructions to make sure all disassembly, maintenance and pack-up procedures are completed.
