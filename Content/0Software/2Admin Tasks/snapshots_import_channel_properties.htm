Import the Channel Properties

  1. Select the check box under the Channel properties heading. Additional check boxes become available.
  2. To import all channel properties, select the Import all check box. This will automatically select all the check boxes.

OR

If you only want to import a subset of the channel properties, select the check boxes for the required elements of the current optical channel. The properties imported for each of the valid check boxes will be displayed in the Optical Channel Properties dialog.

Data Logging properties are not available for import. However, Break properties are included with the Processing properties.

When zones are imported, they will replace all existent zones. Any stream overrides defined for a zone, or a suppression zone, will be maintained only if the associated stream is imported as well.
