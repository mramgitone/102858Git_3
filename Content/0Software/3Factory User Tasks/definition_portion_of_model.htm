Definition of the Portion of the Model

  1. Select the Optics Tab and Data Capture Tab.

  2. View the values for the Pulse Width, Input Voltage, and EDFA1 and EDFA3 settings.

  3. In the Noise Floor Bounding Tab select values for the calibration that are close to and bracketing the optics properties that are actually being used.

Example

If you use an EDFA1 value of say 85mA, you canset the EDFA1 range in the calibration tab to use a min of 80, a max of 90, and a step size of 5. The pulse width and input voltage do not need to be bracketed but can be set to the expected values.

  1. To do this, disconnect the system from the deployed and put in its place a spool of a similar length. It should be at least as long as the deployed .
  2. Find as quiet a location as possible to place that spool. The more noise there is while the data is collected, the less accurate the model will be.
  3. Collect the training data. A full sweep of the system can take up to many hours or even days. Do enough to capture a portion for the model of interest.
  4. In the Noise Floor Bounding Tab select values for the calibration that are close to and bracketing the optics properties that are actually being used. See the example above.
  1. Once done and everything is as quiet as you can get it, select the Collect training data button. This will run for some time - roughly 16s for each combination - and the message below will be displayed.

  1. Once the data has been collected, the model then needs to be trained. In the factory calibration this may be done many times, but for the field calibration you should just do it once and use the same parameters as was used to collect the training data.
  1. After the field calibration has been completed, the Model trained & ready option in the Noise Floor Bounding Tab displays a green light. At this point you need to switch the back to LiveFibre Mode and view the Noise Floor display.

The noise floor bounds can not be as close to the actual noise floor, as if they would have been if calibrated at the factory. Turn on the Use exponential fit option and give it a larger than normal Bounds offset reading. Even though the combination of these two settings can not guarantee an accurate reading, it is better than not having the bounding at all.
