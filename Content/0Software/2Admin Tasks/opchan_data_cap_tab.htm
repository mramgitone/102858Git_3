Data Capture Tab

Data Capture properties allow the user to configure what data the system will acquire, and how that data is acquired. See the image and check the values against the instructions below:

Data Capture Tab

  1. Check the Monitor Start/Monitor End values. If you need to restrict the monitored zone to a subset of the total sensor length, enter a positive value in the Monitor Start field. This will restrict the data that is digitized to only include data after this distance.

If you need to ignore the end portion of the sensor, set the maximum value for the Monitor End field to the physical length. For more details about why these field values can be changed, refer to your training materials.

  1. The recommended value for the Sample Rate is 150 mega samples per second (MSPS). However the range can be between 150 mega samples per second (MSPS) and 180 MSPS. Any reduction or increase will affect the compute load. To change the value, select a rate from the drop-down menu.
  2. The Number of Samples is calculated from a combination of the Start Monitor/End Monitor and Sample Rate fields.
  3. The Input Voltage value maximizes the amplitude of the raw shot without clipping data in the region of interest. The maximum amplitude of the analog voltage can vary between 200 mV and 4 V. from the start of the sensor.

If you use the Raw Display Mode, you can view this raw shot response. The default scale of the Raw display should already be set to 32,767, as this is the maximum value of the digitizer card. From here, the Input Voltage should be set as low as possible without clipping the raw shot in the zone of interest. A “Clip” occurs if any data sample along the sensor has amplitudes in excess of 32,767. If a “Clip” occurs, increase the Input Voltage in the drop down list. The optimal raw shot will have maximum amplitude, without any change to data samples within the region of interest.

  1. Check the Shot normalization check box is selected if you want to normalize all shots to a constant value. Shot normalization is not enabled by default. If enabled, it can reduce the random shot-to-shot amplitude of the raw shots, which improves the system’s SNR. Shot normalization can extend over the entire monitored section, so you can restrict operation to a subsection of the sensor, where there are little or no disturbances. To change these values:
    • To restrict shot normalization to a subsection of the sensor, enter distances that are greater than those defined in the Monitor Startfield, and less than the Monitor End field values, entered in Step 1.
    • Select the tick box. This will make the Shot Normalization Window settings active. You can also select the green and yellow icon on the right-hand side.
