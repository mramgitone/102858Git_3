Graphical Definition of a Zone's Extent

  1. Click-and-drag the zone handles to the desired start and end positions for the newly created zone.
  2. Enter the start and end distances, either textually or graphically.
  3. Select the Submit button, to confirm the creation of the new zone. The zone will be added to the Zone dialog, and the zone icon will be displayed in the Graph display, both are shown below.
  4. Repeat this procedure for each new zone to be defined on the Helios Web Interface.

DO NOT let zones overlap on the sensor. If this happens, edit the new or current zone to remove the overlap. Refer to How to resolve an Overlap.
