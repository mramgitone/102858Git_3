Add/Remove a new Colour Control Point

  • To add a new colour control point, select the inside-right of the colour map. This will add a new colour control point at that amplitude, and with its initial colour set to an interpolation between it's two surrounding points.
  • To remove a colour control point from the current colour map, select the icon next to the Colour Control Point.

If you select in the ruler area, it will have no effect when you want to change the Control Points. If you select within the Colour Bar area, it will only create temporary highlights, not control points.

Edit or Adjust a Colour Control Point

If a control point is edited, this changes the amplitude and/or its colour. You will also see the interpolated colours in the colour bar change to match the setting.

The colour remains unchanged.

  1. Add a new Colour Control Point, as described above.
  2. You can move the Colour Control Point to a new location. To do this, select the white arrow (on the left edge of the colour control point) and drag the Colour Control Point up or down the colour bar, as required. As you do this you will see the amplitude written in the control, change its value to match its current position. A hover gesture appears if you move your mouse over the white arrow, which indicates you can move the arrow. To change the colour of a Control Point, refer to Change the Colour Control Point.

Add a Colour Highlight

The colour highlight cannot be resized once it has been added to the colour bar.

Colour highlights are very useful to dynamically change the colour map to focus attention on a specific range of intensity levels. A Highlight is temporary and does not affect the Colour Map base itself. This allows the Highlight to be created easily and deleted. It is interactive, can be created quickly and you can drag it up and down to see its affects in the Shot and Waterfall displays. This allows you to interactively search for specific amplitudes within these displays and is a powerful way to search without having to change the colour map itself. To create a dynamic colour highlight:

  1. Move the mouse pointer directly over the colour map to the start to get the hover gesture.
  2. Then select-and-drag the mouse to the end position, then release. The highlight colour is displayed over the colour map. It will grow and shrink in size as the mouse is dragged up and down the colour map.
  3. Once a colour highlight has been added, it can be dragged anywhere in the colour map. Select-and-drag the colour highlight to the new location.

Each time you complete a drag you will see the appropriate amplitudes in the Graph and Waterfall displays reflect the movement.

Yellow is the default colour but the highlight colour can be chosen in the Preferences... dialog.

Remove a Colour Highlight

To remove a colour highlight, select the button inside the upper right corner of the colour highlight.

Save a New Colour Map

To save a new Colour Map, select the icon in the Colour Map dialog. This will save the current Colour Map as the default, and can be retrieved from the database if required.

The temporary highlight is not considered part of the colour map and is not stored.

Reload a Saved Colour Map

To restore a colour map that was saved to the database, select on the icon in the Colour Map dialog. This will replace the current Colour Map with the one that was last saved in the database.

The temporary highlight is not considered part of the colour map and is not removed when the Colour Map is reloaded.

Restore the Default Colour Map

This feature is useful if you have edited the Colour Map and want to revert back to the original settings.

The default Colour Map is defined when the software is installed on a Controller. To restore it, select the icon in the Colour Map dialog. The default Colour Map can then be edited, saved and restored as described in the above sections.

Configure Alarm Detection Thresholds in Streams

To determine an optimal Alarm Detection Threshold, use a colour bar with a colour discontinuity at a specific position in the colour map. Below are instructions to configure the Alarm Detection Thresholds in the displayed streams:

  1. Refer to Add/Remove a new Colour Control Point for instructions to add a new Colour Control Point.
  2. Configure the colour map to display from dark blue to a lighter blue on the bottom half.
  3. Repeat this process to set the top half of the colour map to go from yellow to red. Use a colour bar like this to make it easy to see when an Alarm Detection Value has crossed the mid-way point of the full range.
  4. With the system live, observe the background noise level and the disturbance of interest. Adjust the Graph display scale so that the amplitude of the required disturbances are above the midway point in the colour bar. (They MUST BE in the yellow to red colour range).
  5. When an Alarm Detection Threshold is displayed in yellow or red, an Event Indicator will be created.

Do not worry too much if the odd noise value crosses into the yellow/red zone, as the indicator clean process will remove random Alarm Indicators that are not part of a valid disturbance.
