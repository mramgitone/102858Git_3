Introduction

() is a web-based application that monitors real-time information from vibration disturbances along the distance of a Helios Distributed Acoustic Sensor installation. Optical s can run up to 40km, and provide information to operatives in a variety of industries. This information can be used independently or with our proprietary alarm management server, Panoptes.

The workstation can be connected directly to the system, or can be connected via a LAN or a WAN. This allows for both local and remote surveillance and control.

Principle of

The Controller takes the raw backscattered light from the sensor due to an acoustic disturbance and transforms it into detailed knowledge about the time, location and type of disturbance. displays the controller’s response to a variety of alarm disturbance types, and can display alarms when a particular disturbance has been detected that meets a number of alarm criteria. The sophisticated disturbance alarm module has been designed to ignore nuisance alarms while it continues to alarm accurately on disturbance types of interest.

supports simultaneous configuration of detection and multiple alarm types that can produce a number of display outputs, designed to allow users to fine tune and optimise the system’s response. A full suite of diagnostic capabilities has been incorporated into to notify users immediately of any irregularities in the system’s operation. System log and parameter files can be easily downloaded to the user’s local computer in order to understand more fully the system’s operational characteristics.

Not only can display the live response of the system, but it can also instruct the Controller to record to disk the raw system response and then have it played back with the use of a Playback module. This feature allows for an accurate fine-tuned set of system parameters and settings that can provide improved alarm detection capabilities.
