Manage Users Properties

Apart from View Only users, all other users are required to login. The first time the Helios Web Interface web server is accessed, it presents a very basic interface referred to as the View Only Interface. From the Manage Users dialog, you can configure and add or remove users and set their access levels. See the instructions below to add or remove a user.

Add a New User

Once logged into Helios, a new user can be added. To do this:

  1. Select the Admin > Manage users… option. The Users dialog is displayed as shown below.

Manage Users Dialog

  1. To add a new user, select the icon.
  2. Enter the user’s name in the Name field.
  3. Enter in lowercase letters, a name in the Username field. After a user is created their username cannot be changed.
  4. Enter a password in the New password field, and then again in the Confirm password field, so that it is associated with the username. If you have to edit a user’s role or name, it is not necessary to populate the password fields.
  5. Select the role for the new user. For description of each role, see below:
    • A View Only user can only start and stop the Helios Controller, but cannot change any properties.
    • An Admin user can perform all the normal operations, but cannot reset factory defaults, manage or delete users.
    • Factory user. This is a reserved login used by Fotech Solution Ltd only.

6. Select the Apply and Save buttons, when all information has been entered, to save the new user into the Helios database.

Remove a User

The default Admin (and Factory) users that come with the system cannot be deleted.

To remove a user:

  1. Select the user’s name and then the icon. A confirmation dialog appears as shown below.
  2. Select the OK button to delete the user, or the Cancel button to abort the deletion process.
