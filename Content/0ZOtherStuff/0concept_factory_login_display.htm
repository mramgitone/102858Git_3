Precautions

Before you wipe the database, check with the person in authority that this should be done and then do a back-up of your system. Once you wipe the database, the information cannot be retrieved.

Instructions to Wipe the Database

 

 Save as factory defaults

Use this function with extreme caution. If a Helios Controller is reset to its factory defaults, all property changes made to the system will be destroyed! This includes all zone definitions and their associated properties.

There may be times when the edited properties no longer produce acceptable results in the Graph. If this happens, the Helios system can be restored to its Factory Defaults. To restore the Helios system to its Factory Defaults:

  1. Select Admin > Reset to factory defaults. A confirmation dialog will be displayed.
  2. Select OK to reset the Helios system to its Factory Defaults. The system will then shut down and restart the Helios Controller.
  3. Select Cancel to dismiss the warning message and return to the Operations Screen.
