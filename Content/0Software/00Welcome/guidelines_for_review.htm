Conditional Tagging

Conditional tags have been added to some sections where users will see differences, depending upon which user level they are granted to log-in. A list of the existing tags is in the image below. One for Panoptes may also be included at some point in the future.

Conditional tags in Flare are used to “Exclude” highlighted sections from a user’s view. A lot of the tags are set so that the View Only user will not see the content. Where more than one level of user shouldn’t see content, a patterned background will be seen e.g. Basic & Admin users background will look speckled (see below).
