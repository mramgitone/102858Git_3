SK: - I don't think the way this is presented is very clear. Specifically the icon with the plus is always used to zoom, the one with the minus is always used to unzoom to the previous display, and the one with the big crossed arrows is always used to pan.

Whether they zoom in depth, time, or both depends on where you drag the mouse, regardless of which icon you have clicked.

Specifically if you click and drag in the fibre shot panel or in the depth ruler, you only zoom/pan in depth since that is the only axis available. And if you click and drag in the time ruler you only zoom/pan in time since that is the only axis available. But if you click and drag in the waterfall display you can zoom/pan in both depth and time since both axis are available

 

Icon Description
If selected, it zooms in on the Graph.

If you select-and-drag in the shot panel or the depth ruler, this icon allows you to zoom and pan in depth only. If you click-and-drag in the time ruler, then you can only zoom in time only. To do this, define the start and end positions then release the mouse button.

Pan mode will return a yellow text field. This will indicate the time difference between the upper most row and current time. A negative value implies that the top row is before the current time.
If selected, it zooms out to the previous display of the Graph. This will reset the display to the previous zoom level. If pressed multiple times it will eventually zoom out to the original view.

Zooms in and pans for distance only in the Waterfall Display because both axes are available. Pan mode can only be used on data that has already been displayed.

To zoom and pan in distance and time, select the start distance-time point in the Waterfall Display. Select-and-drag the pointer to the distance-time end point and release the mouse button.

It also pans out if selected, and will return to the previous display.
