Import Alarm Reports

To import all of the Alarm Reports:

  1. Under the Alarm Reports header, select the All check box and then the Submit button. This will automatically select all the Alarm Reports, as well as all of the components of each Alarm Report.
  2. To import a single Alarm Report, select the required Alarm Report and then the Submit button.
  3. Next you need to import the Alarm Report components: Select the desired components and then the Submit button. The Import module allows for either an entire database to be restored, or just a single Alarm Report component.

If an Alarm Report is selected, the relevant streams and other alarm report components associated with it, via High Level Rules, are included. They are mandatory for an import to be successful. Likewise, when a Stream is selected, the associated Detection Statistic template and Indicator mask are also selected.
