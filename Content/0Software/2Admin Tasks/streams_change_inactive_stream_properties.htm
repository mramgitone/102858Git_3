Change Inactive Stream Properties

  1. You can change properties for an inactive stream, but they will not take effect until the stream is made active. To do this, select the associated Alarm Report, for the stream. This can be done in the Optical Channel Properties dialog. Refer to How to activate an Alarm Report.
  2. Once the stream is selected, the individual properties can be overridden. To override an individual property, deselect the Default check mark. This makes the associated field editable.
  3. Change the properties and then select the Apply and Save buttons. The system then stops, and restarts with the new properties.
