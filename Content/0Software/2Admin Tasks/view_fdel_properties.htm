View FDEL Properties

The current system facilitates complete user interaction with the Helios system via a web interface and the underlying application still uses a properties file to keep a record. It is possible to view the actual contents of the properties file via the web interface. To do this:

  1. Select Admin >View FDEL properties. A new dialog that displays the key-value pairs of the systems properties file, ordered alphabetically by key. The image below shows an example of this window.

Example of the FDEL Properties Diagnostics Dialog

  1. Scroll up or down to view all of the properties contained in the file.
  2. Select the Refresh button to request the latest list of properties from the Helios system. This dialog ONLY reads the properties from the moment it is selected. This file is updated after every successful properties update, when the Apply and Save buttons are used. It is possible that the properties viewed on the screen can be out of date at the time it is viewed!
  3. To close this dialog, select the icon.
