Change Start and End Distances in a Zone

To change a zone’s start and end distances:

  1. Select the [zone resize] icon. This displays the start and end drag handles with the zone’s current measurements. These can then be changed, graphically or manually.

The same procedure for Define a New Zone can be used for this.

  1. Select the Submit button to confirm the change(s). The image below displays the interface to edit the start and end distances of a current zone.
