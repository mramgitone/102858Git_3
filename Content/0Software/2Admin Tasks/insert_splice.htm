Insert a Splice

  1. Select the Insert a Splice… button

Splice Dialog

  1. Enter the exact position of the splice, and the amount of that was inserted into the sensor cable.
  2. Select the Save button. You can also cancel or reset the values from this dialog.
  3. To check the splice is positioned, select the Splice History... button. If you choose the Refresh button, the system will update with the latest real-time information. To exit the dialog, select the Close button.
