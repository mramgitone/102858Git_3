Zone Properties

Zones are a very powerful feature of the Helios system because they enable an Operator to monitor set distances along the sensor. For more information, refer to your training materials.

Zones are user-defined sections of the sensor that can be created easily and manipulated to be or have:

  • Edited to change the Start and End distances of the zones already defined.
  • Unique processing properties applied to them. This is on a per stream basis, which allows them to process data differently than the global sensor properties.
  • Logged and stopped instantly.
  • Zoomed into and out of, directly.
  • Deleted.

Below, in more detail, are the tasks you can complete with Zones.

Define a New Zone

  1. To access the Zones dialog, go to Admin > Zones… . If there are no zones defined, you will see the dialog below.

Zones Properties Dialog displayed without zones defined

  1. Enter a name for the new zone, in the dialog box. An example is given below.

Enter Zone Name prompt

  1. The Graph display will update to show two drag handles, the left for the start distance and the right for the end distance of the new zone. See the image below. You can either:
    • Drag the handles to the desired locations.
    • Enter the exact distances into their respective text fields that are located either side of the drag handles.
    • Define the zone’s extents graphically. (Refer to Graphical Definition of a Zone's Extent.)

Definition of a Zone in the Graph

Edit a Zone

There are six icons displayed to the right of the zone name that permit direct manipulation of the zone. These are:

  • Change Start and End Distances in a Zone.
  • Change the Properties of a Zone.
  • Log a Zone dynamically.
  • Termination of a Zone Logging Operation.
  • Zoom in/out of a Zone.
  • How to resolve an Overlap.
