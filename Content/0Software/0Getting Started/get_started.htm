Get Started

To set-up and operate the Helios System, it is advised that you attend a training course and at least know how to log data. If you need to book a training course, contact Customer Support. If necessary, refer to your training materials for more in-depth reasons why you have to complete each step.

Below are instructions to set-up and start to use the Helios System. It is recommended that you read the whole section before you start.

Get Started for Factory Users

  1. Have you checked you can use the system? Refer to Pre-requisites of Use and read the Guidelines for the use of the Help File.
  2. Refer to the Installation & Configuration Guide and Post-installation Instructions to check the system is set-up and ready to use.
  1. Log-in to the software. You will see the The View Only Display.
  2. Refer to the What’s New to see what the new functionality is for this release.
  3. Set up the Helios Software. Check the default settings.
  4. Make sure the mains switch is turned on.
  5. If you are a Factory User, you will need to complete instructions for How to Set-up the Helios Software.
  6. Turn the Helios Unit on.
  7. is a web server-based system. The data processing will start automatically each time the system starts up. Press the start button on the front of the Helios Unit to initiate data processing, then go to the Helios Controls and Change the Monitoring Status. The options will turn on and the buttons will illuminate in red.
  8. Complete the Operations Set-up instructions.

You will need to have System Administrator level access to the system, as these steps are normally completed by a Fotech Engineer.

Contact Customer Support if necessary, then complete the Operations Set-up instructions.

Get Started for Administrator Users

To use the Helios System, it is advised that you attend a training course and at least know how to log data and troubleshoot the system. If you need to book a training course, contact Customer Support. If necessary, refer to your training materials for more in-depth reasons why you have to complete each step.

Below are instructions to start to use the Helios System. It is recommended that you read the whole section before you start.

  1. Have you checked you can use the system? Refer to Pre-requisites of Use and read the Guidelines for the use of the Help File?
  2. Log-in to the software. You will see the Operations Overview Screen.
  3. Refer to the What’s New to see what the new functionality is for this release and then start to use the system.

Get Started for View Only Users

To use the Helios System, it is advised that you attend a training course and at least know how to view logged data and disturbances shown by the system. If you need to book a training course, contact Customer Support. If necessary, refer to your training materials for more in-depth reasons why you have to complete each step.

Below are instructions to start to use the Helios System. It is recommended that you read the whole section before you start.

  1. Have you checked you can use the system? Refer to Pre-requisites of Use and read the Guidelines for the use of the Help File?
  2. Open your browser and enter the IP address for the software. You will then see the The View Only Display.
  3. Refer to the What’s New to see what the new functionality is for this release and then start to use the system.

If at any time during this procedure you need to log-on/off. Refer to How to Login and How to Log out.
